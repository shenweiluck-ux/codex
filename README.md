# LTC 2.0 智能销售作战 Demo

这是一个基于“LTC2.0 智能销售作战与资源加速系统设计”方向搭建的无依赖前端 demo。当前版本聚焦原型验证：把销售输入转化为商机评分、风险预警、动态任务和资源加速方案。

## 功能

- 销售实时录入客户、阶段、金额、赢率、关键情况和下一步计划。
- 规则引擎生成商机健康度评分和评分解释。
- 自动识别预算、竞争、决策链、交付和资源诉求等风险/机会关键词。
- 按 LTC 阶段推荐下一步作战任务。
- 自动生成资源申请摘要。
- 管理者驾驶舱统计管道金额、平均赢率、高风险商机和资源需求。
- 使用浏览器 localStorage 保存最近 10 条跟进记录。


## 不懂技术怎么查看成果

如果你只是想看当前 Demo 页面，请看这份说明：

```text
docs/how-to-see-demo.md
```

最简单的方式是进入 `frontend` 文件夹，然后双击 `index.html`，浏览器会直接打开 Demo 页面。

## 运行

```bash
npm start
```

然后访问：

```text
http://localhost:3000
```

也可以直接打开：

```text
frontend/index.html
```

## 检查

```bash
npm run check
```
