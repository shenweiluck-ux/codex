# LTC 2.0 Demo Scope

本 demo 基于“LTC2.0 智能销售作战与资源加速系统设计”文档名称、GitHub README 摘要和用户“先做 demo”的要求，先实现无需依赖的规则版原型。

## Demo 范围

- 销售录入客户、阶段、金额、赢率、关键问题和下一步动作。
- 根据金额、赢率、阶段、风险关键词和资源诉求计算商机健康度。
- 输出风险预警、作战任务、资源加速建议和管理者驾驶舱指标。
- 使用浏览器 localStorage 保存输入历史，方便演示连续跟进。

## 非本期范围

- 登录和权限。
- 数据库持久化。
- 真实 CRM/ERP 集成。
- 大模型 API 调用。
